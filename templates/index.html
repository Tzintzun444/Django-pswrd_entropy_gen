{% trans "Password Entropy Generator" %}

{% trans "What is this project?" %}

{% blocktrans %} This password generator was originally created as a way to apply Python knowledge, but it gradually evolved into a technical, simple, and flexible tool with a clear purpose: to produce passwords that are reliable, strong, and resistant to being cracked. {% endblocktrans %}

{% blocktrans %} It’s not meant to generate easily memorable passwords, but to offer high-security credentials—especially valuable in systems that demand maximum protection, such as sensitive credentials, environment variables, access keys for servers, or APIs. {% endblocktrans %}

{% trans "Why trust entropy?" %}

{% blocktrans %} The strength of a password isn't about how “complicated” it looks—it's about how unpredictable it really is. That unpredictability is measured using a core concept from information theory: entropy. In this project: {% endblocktrans %}

  • {% blocktrans %}Each password includes a real entropy value in bits.{% endblocktrans %}
  • {% blocktrans %}It estimates how many attempts per second would make a hacker to crack it by brute force.{% endblocktrans %}
  • {% blocktrans %}It calculates a decryption time, offering a reliable reference for how hard it is to break.{% endblocktrans %}

{% blocktrans %}This gives you a real, data-based understanding of your password’s resistance—far more meaningful than a generic strength meter.{% endblocktrans %}

{% trans "Design philosophy" %}

{% blocktrans %} This tool doesn’t rely on commercial standards of password complexity. Instead, it offers: {% endblocktrans %}

  • {% blocktrans %}Generation free of human bias.{% endblocktrans %}
  • {% blocktrans %}Full control over configuration and strength.{% endblocktrans %}
  • {% blocktrans %}A mathematical foundation behind each password{% endblocktrans %}
  • {% blocktrans %}Adaptability to a wide range of environments (development, production, CI/CD, etc.){% endblocktrans %}

{% blocktrans %}The generator is flexible by design, allowing each password to suit the security level required for its context.{% endblocktrans %}

{% trans "Project purpose and philosophy" %}

{% blocktrans %} This project was created with a practical and educational spirit—starting as a personal exercise in Python and growing into a tool meant for developers, sysadmins, and anyone seeking real security without unnecessary complexity. {% endblocktrans %}

{% blocktrans %} It is built to be Open Source, freely shareable and continuously improvable. Its purpose is to promote solid practices, protect sensitive systems, and offer a solution rooted in theory rather than guesswork. {% endblocktrans %}
